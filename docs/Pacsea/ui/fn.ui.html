pub fn ui(f: &mut Frame<'_>, app: &mut AppState)
Expand description

Render a full frame of the Pacsea TUI.

This function is the single entry point for drawing the interface and is meant to be called each tick or after state changes.

Arguments:

  • f: ratatui frame to render into
  • app: mutable application state; updated during rendering for selection offsets, cursor position, and clickable URL geometry

Behavior summary:

  • Applies the global theme and background
  • Ensures the results selection remains centered within the list viewport by adjusting the list stateâ€™s internal offset
  • Renders the top results list with source badges and install markers
  • Renders the middle row with Recent, Search (with visible cursor), and Install panes; titles and colors reflect focus
  • Computes and stores a clickable rectangle for the details URL when it is present, enabling mouse interactions handled elsewhere
  • Shows a help footer with keybindings inside the details area
  • Draws modal overlays for network alerts and install confirmations
