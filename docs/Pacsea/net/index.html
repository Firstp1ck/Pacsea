Network and system data retrieval for package metadata.

This module fetches package details from two sources:

  • AUR RPC v5 endpoints for AUR packages
  • Arch Linux official package JSON pages and local pacman -Si for official repository packages

It provides helpers that normalize data into PackageDetails and utilities to search the AUR. For robustness and offline support, official details are attempted via pacman -Si first, then fall back to web JSON.
