Expand description

Spawn a background task to refresh the official index and notify on changes.

On success, merges new names while preserving previously enriched fields (repo, arch, version, description) for packages that still exist. Persists the updated index and sends a unit message on notify_tx when the set of names changes. On failure, sends a human-readable error on net_err_tx.
